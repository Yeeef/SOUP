# README

zju compiler project

## requirments

- 用 lex 写出一个 tiger 语言或者类 C 或者类 PASCAL 某个语言的词法分析器
- 用 YACC 的 分析方法完成语法分析，并生成语法树和中间代码
- 如果生成目标代码, 可加分
